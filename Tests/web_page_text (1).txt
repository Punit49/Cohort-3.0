Which statement is correct?
A. Function declarations and function expressions are exactly the same.
B. Function declarations are hoisted, while function expressions are not hoisted in the same way.
Correct Selection
C. Function expressions are hoisted, but function declarations are not.
D. Neither is hoisted.
2
Which statement is technically correct?
A. Parameters and arguments are the same thing.
B. Parameters are values passed during a function call.
C. Arguments are placeholders in a function definition.
D. Parameters are placeholders, while arguments are actual values passed to the function.
Correct Selection
3
Which situation is generally NOT recommended for arrow functions?
A. Utility/helper functions
B. Single-expression functions
C. Object methods that rely on this
Correct Selection
D. Callback functions
4
When is a default parameter value used?
A. When any falsy value is passed.
B. Only when the argument is undefined or omitted.
Correct Selection
C. When null is passed.
D. When an empty string is passed.
5
Which statement about rest parameters is true?
A. Rest parameters must always be the first parameter.
B. A function can have multiple rest parameters.
C. Rest parameters collect remaining arguments into an array.
Correct Selection
D. Rest parameters store arguments as an object.
6
Which statement about return is true?
A. A function can execute multiple return statements in a single call.
B. return pauses a function and resumes later.
C. Once a return executes, the function immediately exits.
Correct Selection
D. return can only return primitive values.
7
Which capability directly results from functions being first-class citizens?
A. Functions can only be declared globally.
B. Functions can be stored in variables and passed to other functions.
Correct Selection
C. Functions automatically become callbacks.
D. Functions cannot be returned from other functions.
8
Every callback function is:
A. Recursive
B. Anonymous
C. Passed to another function for later execution
Correct Selection
D. A higher-order function
9
Which statement is true?
A. Every callback is a higher-order function.
B. Every higher-order function is a callback.
C. A function that accepts another function or returns one is a higher-order function.
Correct Selection
D. Higher-order functions must be recursive.
10
Why are anonymous functions commonly used as callbacks?
A. They execute faster.
B. They reduce memory usage.
C. They are often needed only once and don't require a name.
Correct Selection
D. JavaScript requires callbacks to be anonymous.
11
Historically, what was the primary purpose of IIFEs?
A. Improving recursion performance
B. Creating private scope and avoiding global variable pollution
Correct Selection
C. Replacing loops
D. Making functions asynchronous
12
Which characteristic is NOT required for a function to be pure?
A. Same input should produce the same output.
B. No side effects.
C. Must use only local variables.
Correct Selection
D. Predictable behavior.
13
A function becomes impure if it:
A. Uses parameters.
B. Uses loops.
C. Modifies external state or depends on changing external data.
Correct Selection
D. Returns an object.
14
Which condition is essential for recursion to terminate?
A. Callback function
B. Base case
Correct Selection
C. Arrow function
D. Rest parameter
15
Which description best represents recursion?
A. Solving a problem by repeatedly creating larger versions of it.
B. Solving a problem by reducing it into smaller versions of the same problem.
Correct Selection
C. Executing multiple functions simultaneously.
D. Repeating code without function calls.
